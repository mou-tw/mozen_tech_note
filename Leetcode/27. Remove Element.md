雙指針解法
思想為設立頭尾pointer，並各自往中間移動，遇到目標值即交換，並增加數值，以達到修改數組列表，並回傳累積值的目的。

python 
```
class Solution:

    def removeElement(self, nums: List[int], val: int) -> int:

        tmp =0

        st, end = 0 , len(nums) -1

        while st < end:

            if nums[end] == val:

                end-=1

            else:

                nums[end] , nums[st] = nums[st] , nums[end]

                st+=1

  

        return len([i for i in nums if i != val])
```
